## Description
This is a simple viewer for Conway's Game of Life, written in java using javafx. The viewer can import patterns from [RLE](https://conwaylife.com/wiki/Run_Length_Encoded) files and show how
they evolve over time, by following the common [rules](https://conwaylife.com/wiki/Conway's_Game_of_Life#Rules).


## Functionality

- import a pattern from a RLE file
- draw on the game grid with your mouse
- move around the grid
- zoom in / out
- pause / resume the playback
- save the current pattern
- reset the game to the last saved pattern
- clear the game grid
